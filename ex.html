Tic-Tac-Toe-Game

A browser-based Tic Tac Toe game using HTML, CSS, and JavaScript. Two players alternate turns marking "X" or "O" on a 3x3 grid. JavaScript handles the game logic, checking for wins or draws. The interface is responsive and visually appealing, providing a simple yet engaging gaming experience for all users.

Certificate-Generator

A ReactJS-based certificate generator that allows users to input details like name, course, and date to create custom certificates. The project dynamically renders the input data onto a certificate template and provides an option to download the generated certificate as a PDF, making it a useful tool for educators and event organizers.

Youtube-Frontend-Clone

A YouTube frontend clone using HTML and CSS replicates the layout and design of YouTube's homepage.
